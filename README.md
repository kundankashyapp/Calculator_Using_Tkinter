# Calculator_Using_Tkinter

A simple GUI calculator built using Python's **Tkinter** library. This calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- User-friendly GUI
- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Clear (C) and equals (=) buttons
- Lightweight and easy to use
